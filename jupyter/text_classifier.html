# Preprocesamiento de texto
def preprocess_text(text):
    # Crear el objeto stemmer asignandole el idioma espaol
    stemmer = SnowballStemmer("spanish")
    # Obtener las palabras vacas en espaol
    stop_words = set(stopwords.words('spanish'))
    # Conveierte el texto a minsculas
    text = text.lower()
    # Tokenizar el texto
    tokens = word_tokenize(text)
    # Se aplica el stemmer solo a las palabras de interes 
    clean_tokens = [stemmer.stem(word) for word in tokens if word.isalnum() and word not in stop_words]
    # Se recontruye la cadena de texto
    clean_text = " ".join(clean_tokens)
    return clean_text

# Funcin para transformar los textos al embebido Word2Vec
def word_averaging(model, words):
    mean = []
    for word in words:
        if word in model.wv.index_to_key:
            mean.append(model.wv[word])
    if not mean:
        return np.zeros(model.vector_size)
    return np.mean(mean, axis=0)

def word_vectorizer(X):
    return np.array([word_averaging(model, words) for words in X])

Conlusiones

  • Se realiz la reduccin de dimensionalidad con svd puesto que es un mtodo para realizar la reduccin de dimensionalidad para matrices dispersas que es ms eficiente que aplicar PCA.

  • Tambin se realiz la aproximacin de reduccin de dimensionalidad con Word2Vec, este mtodo puede manejar la semntica de las palabras, pero al realizar la clasificacin con SVM tuvo menor rendimiento a la reduccin con truncate SVD, casi de 10% menor.

  • Se compararon los modelos de regresin logstica y SVM, para elegir el que mejor exactitud arroje con los datos, se relizan pruebas con vectorizadores diferentes y el SVM arroja mejores resultados para los datos, quedando con un 0.87 de exactitud. SVM es una tcnica clsica pero para este caso de uso fue efectivo combinandolo con vectorizadores como TF-IDF, adicional a esto maximiza la distancia entre el hiperplano de separacin de los puntos ms cercanos a esta.
